transform

  1. 이동옵션 : translate(x축, y축); , translateX(단위), translateY(단위)
  2. 회전옵션 : rotate(Ndeg); , rotateX(), rotateY(), rotateZ()
  3. 크기옵션 : scale(x축, y축), scaleX(), scaleY() (단위 배수)
  4. 모서리 각도 조정 : skew(Xdeg, Ydeg), skewX(), skewY()
  5. 행렬 : marix(scakeX(), skewY(), skewX(), sclaeY(), translateX(), translateY())

transition

  • transition-duratio : 변환되는 속도를 지정
  • transition-delay : 변환대기 시간
  • transition-property : width height
  • transition-timing-function :
    • linear : 처음부터 끝까지 같은 속도
    • ease : 처음, 끝을 부드럽게 (기본값)
    • ease-in : 처음을 느리게
    • ease-out : 마지막을 느리게
    • ease-in-out : 처음 마지막을 느리게
    • cubic-bezier(n, n, n, n,) : 속도를 직접 셋팉
  • transition : property duration timing-function delay
